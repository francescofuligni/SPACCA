# ProgettoSPACCA

Gioco di carte SPACCA

1) da Main menu è possibile accedere a schermata di autenticazione per admin e a schermata per entrare in una partita precedentemente creata
2) una volta autenticato è possibile creare una partita singola o un torneo al quale verrà associato un codice
3) ad ogni partita verrà richiesto di aggiungere gli user dei partecipanti
4) essi potranno accedere inserendo il loro user e il codice per la specifica partita
